# Type Inference

## Type Inference in few words

## Type Inference in details

[[Type Inference]] is the thing that allows you to not have to specify the [[Type]], the [[Type]] will be inferred by the literal, e.g.:

```java
var i = 3; // int
var d = 3.0; // double
var c = 'c'; // char
var s = "c"; // string
```
