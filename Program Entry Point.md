# Program Entry Point

## Program Entry Point in few words

## Program Entry Point in details

The start of a [[Kotlin]] Application is the `main` [[Function]].

```kotlin
fun main() {
}
```

We can accept `String` arguments !

```kotlin
fun main(arguments: Array<String>) {
	...
}
```
