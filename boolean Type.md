# boolean Type

## boolean Type in few words

## boolean Type in details

`boolean` needs 1 byte to express `true` or `false`, there no [[Implicit Conversion]] from one type to `boolean`

```java
boolean b1 = 2;     // Error
boolean b2;         // b2 is false
boolean b3 = true;
boolean b4 = 3 > 4; // b4 is false
```
