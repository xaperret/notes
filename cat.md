# cat

## cat in few words

The command `cat` allows us to concatenate one or more files into the [[Standard Output]].

## cat in details

### Exercises

- Use `cat` on file `/etc/passwd` to get your user information
  - `cat /etc/passwd`
- Do the same for `/etc/group`
  - `cat /etc/group`
