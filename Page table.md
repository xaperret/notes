# Page table

## Page table in few words

A page table exists for each [[Process]].

Each table is maintained by the [[Operating System]] and used by the [[Memory Management Unit]].

An entry will have:
- Page number
- Page size
- Access permission
- Bit *valid page* or *page present in RAM*
- Bit *dirty page*
