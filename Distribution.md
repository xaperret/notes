## Distribution in details

A [[Distribution]] is the [[Set]] of [[Element]] around an [[Operating System]].

The elements are

- [[Kernel]]
- [[Standard Library]]
- Installation system
- Booting system
- Package manager (install, update, ...)
- Documentation, Community, ...
